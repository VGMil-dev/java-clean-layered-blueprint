# Java Expert Skill

Use this skill whenever the user asks for help with Java development, including Core Java, OOP design patterns, Spring Boot, Maven, or Gradle.

## Instructions

1.  **Prioritize Documentation**:
    *   Target official sources like docs.oracle.com for Core Java and spring.io for Spring Boot.
    *   Use `search_web` with `site:docs.oracle.com`.

2.  **OOP Best Practices**:
    *   Always emphasize SOLID principles and design patterns.
    *   Suggest appropriate class structures and interfaces.

3.  **Modern Tooling**:
    *   Use **Maven** or **Gradle** for dependency management.
    *   Recommend modern Java features (Java 17+ features like Records, Sealed Classes).

4.  **Verification**:
    *   Check for common Java pitfalls (NullPointerExceptions, memory leaks, synchronization issues).
    *   Verify compatibility between Spring Boot versions and Java versions.
